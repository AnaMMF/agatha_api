# Instrucciones para importar la base de datos de Agatha

1. Abrir phpMyAdmin (o la herramienta que el profesor utilice).
2. Crear una base de datos nueva llamada `agatha`.
3. Ir a la pestaña “Importar”.
4. Seleccionar el archivo `agatha.sql`.
5. Aceptar.

La base de datos tendrá:
- usuarios de prueba
- historias de ejemplo
- registros de inactividad
- palabras y lugares para los retos
